# Source and license notes

The primary data source is CEPII-BACI. Users should cite CEPII-BACI and Gaulier & Zignago (2010) when using the source data. The original BACI data source should be downloaded from CEPII according to CEPII's access and licensing terms.

This repository is intended as a BACI-derived benchmark layer for variational inequality and MARL experiments. It does not replace the original BACI database.

Suggested dataset-package license for the code: MIT License. Suggested license for processed benchmark data: follow the source-data attribution requirements and use an open data license compatible with CEPII-BACI terms when publishing on Zenodo.
